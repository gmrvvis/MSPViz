MSPViz

This is a visualization tool for the data of the model of structural plasticity. The data musb be in JSON format, the user can use the MSPJSONGenerator.py script to transform the data, and be able to load in the visualization tool.

All the views lets the use of:

  • Zooming: Using the mouse wheel.
  • Panning: Click on the background and moving the mouse.
  • Selection: Pressing shift button and clicking over the neuron representation. Additionally, the macroscopic view lets multi selecion, pressing shift key and moving the mouse over the view.

Main menu:

  • File/Open local simulation: Load the simulation files from local directory. Needs the 3 standart files, called "LocalNeuronInformation.json", "GlobalSimulationParams.json" and "Connectivity.json". The user must select the three files at the same time.
  • File/Reset Visualization: Reset the visualization tool, to prepare a new visualization.
  • Edit/Take an snapshot: Take an snapshot of the current view and at the current simulation step. (Not work propertly)
  • Edit/Preferences: Lets the user to change the preferences of the visualization.
  • View/Global Connections View: This view lets the user to analyze the total connections in the simulation, such as the total number of synaptic elements.
  • View/Macroscopic View: This view give the user and overview of the rewiring proces during the simulation. Neurons are represented as trinagles for excitatory neurons, and circles for inhibitory neruons, the color of each shape is acording to its calcium concentration. The arrows are the conections, and the color of the arrows are the type of the connection (EE, EI, IE, II).
  • View/Microscopic View: This view shows the relation of synaptic elements of the neurons, previously selected in the Macroscopic View.
  • View/Detailled Microscopic View: This view shows the exact number of synaptic elements of the selected neuron from the Microscopic View.
  • Help/About: Information about this tool.

Bottom controls:

  • Simulate button: Start the visualization of the simulation.
  • Stop simulation button: Stop the visualization of the simulation.
  • Timeline: Lets the user go to an especitfic step of the simulation.
  • Step: Lets the user go to an especitfic step of the simulation.
