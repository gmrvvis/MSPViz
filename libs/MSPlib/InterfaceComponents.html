MSPViz

MSPViz is a visualization tool for Structural Plasticity Models. It uses a novel visualisation technique based on the representation of the neuronal information through the use of abstract levels and a set of representations into each level. This hierarchical representations, lets the user interact and change the representation modifying the degree of detail of the information to be analysed in an simple and intuitive way, through the navigation of different views at different levels of abstraction. The designed representations in each view only containing the necessary variables to achieve the desired tasks, avoiding overwhelming saturation of information. The multilevel structure and the design of the representations constitutes an approach that provides organized views that facilitates visual analysis tasks. Moreover, each view has been enhanced adding line and bar charts to analyse trends in simulation data, and also filtering and sorting capabilities can be applied on each view easing the analysis. Additionally, some other views, such as connectivity matrices and force directed layouts, have been incorporated, enriching the already existing views and improving the analysis process. Finally, this tool has been optimized to lower render and data loading times, even from remote sources such as WebDav servers. The data musb be in JSON format, the user can use the MSPJSONGenerator.py script to transform the data from NEST to be able to load in the visualization tool.

All the views lets the use of:

  • Zooming: Using the mouse wheel.
  • Panning: Click on the background and moving the mouse.
  • Selection: Pressing shift button and clicking over the neuron representation. Additionally, the macroscopic view lets multi selecion, pressing shift key and moving the mouse over the view.

Main menu (on the left): The main menu are structured into thre different parts. The first one is on the top, where the user can select different views (at different levels of abstraction). These views are:

  • Global Connection: This view shows the global connectivity data in the simulation, that is the total number of synaptic elements and connections.
  • Macroscopic: This view is designed to have a global view of all neurons and synapses in the simulation, and at the same time, this view shows the connections between neurons in each time step of the simulation.
  • Connectivity Matrix: This view shows the synaptic connection of neurons during the simulation in a Matrix way.
  • Microscopic: This view allows to analyze and compare the synaptic elements in each neuron.
  • Detailed Microscopic: This view is designed to visualize directly the number of synaptic elements in each neuron, showing in each time step how the synaptic elements appears and disappears in a certain neuron.

After the view selection, there is the main controls of the application, related with the most common tasks during the analysis.

  • File menu: Load the simulation files from different places, such as local simulation or simulations stores on remote serves, following simple attps request or WebDav protocols. In this menu, the user could store and load different configuration related with the visualization, such as color paletes for each parameter under analysis.
  • Color preferences menu: In this menu, the user can select different and standar color paletes for each parameter under analysis, such as the calcium concentration, the type of connection or the type of the neuronal element.
  • Global configuration menu: This menu lets the user to configure some generic visual parameters for the visualization, such as type of connection, the type of the synaptic elements to visualize or the speed of the visualization.
  • Global configuration menu: This menu lets the user to configure some generic visual parameters for the visualization, such as type of connection, the type of the synaptic elements to visualize or the speed of the visualization.
  • View configuration menu: This menu lets the user to configure specific parameters of the active view, for example, wich connectios to show on the Macroscopic View or the number of historam steps on Detailed Micoscopic.
  • Filter menu: This menu lets the user to filter the data within a specific visualization, selecting what data is displayed easing the analysis.
  • Sort menu: This menu lets the user to sort the data within a specific visualization using different criterias such as calcium concentration or number of connections.

On the bottom part of the main menu there are two generic operations

  • Download view image: To take high quality screenshots of the visual representaion of the data.
  • Help/About: Information about this tool.

Bottom controls:

  • Play Simulation: Start the visualization of the simulation.
  • Stop Simulation: Stop the visualization of the simulation.
  • Timeline: Lets the user go to an especitfic step of the simulation.
  • Step: Lets the user go to an especitfic step of the simulation.
  • Layouts: The different views have different layouts to place the elements on the screen following different paradigims. The layouts depends on the view under analysis.
